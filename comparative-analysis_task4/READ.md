# Comparative Study of Bioinformatics Databases: NCBI, UniProt, and PDB


This project compares three major bioinformatics databases: **NCBI, UniProt, and the Protein Data Bank (PDB)**. These databases are widely used for storing and analyzing biological sequence and structural data.


### NCBI

Provides access to nucleotide sequences, genomes, and literature. It includes tools like BLAST for sequence comparison.

### UniProt

A protein database that contains curated information about protein sequences, functions, and annotations.

### PDB

Stores 3D structures of proteins and nucleic acids, which can be visualized using tools like PyMOL.


## Key Differences

* **NCBI:** DNA/RNA sequences + alignment tools
* **UniProt:** Protein function and annotation
* **PDB:** 3D molecular structures


## Applications

* Sequence analysis and comparison
* Protein function prediction
* Structural visualization
* Drug discovery research
* Evolutionary studies
