# Bioinformatics-codealphatasks

This repository contains a collection of introductory bioinformatics analyses performed as part of my training.

## Tasks Included

* BLAST Analysis
* Primer Design
* Multiple Sequence Alignment (MSA)
* Comparative Analysis

Each folder contains:

* A report (PDF)
* Screenshots of results
* A brief explanation of the workflow

## Tools Used

* NCBI BLAST
* Clustal Omega
* AlphaFold DB
* PyMOL

This repository demonstrates foundational skills in sequence analysis and interpretation.
