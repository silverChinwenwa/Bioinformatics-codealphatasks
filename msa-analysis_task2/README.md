## Multiple Sequence Alignment (MSA)

### Objective

To align multiple biological sequences in order to identify conserved regions, similarities, and evolutionary relationships.

### Tools Used

* Clustal Omega 

### Method

* Protein/DNA sequences were obtained from a public database, NCBI
* Sequences were input into Clustal Omega for alignment.
* Default alignment parameters were used.
* The aligned output was analyzed for conserved regions and sequence variations.

### Results

* Conserved regions were observed across multiple sequences, indicating functional or structural importance.
* Some regions showed high variability, suggesting evolutionary divergence.
* Alignment gaps were introduced to optimize similarity across sequences.
